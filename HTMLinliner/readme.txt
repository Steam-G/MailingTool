Проект объединяет HTML и CSS в единый файл, что позволяет использовать такой файл для отправки электронных писем. 
Для работы сперва надо запустить программу, в корневом каталоге появятся папку HTML и HTMLinliner
В папку HTML надо скопировать HTML-документ с папкой css и запустить программу.
По окончании работы в папке HTMLinliner появятся собранные файлы.
